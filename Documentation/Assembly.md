# Assembly

## Part list

Please check the part list in the [BOM section](Shopping_list.md)

## Tool list

Check that you have all the tools you need:

- A set of screw drivers and wrenches for hexagon screws (M3, M5 and M8)
- A set of allen keys
- Pliers
